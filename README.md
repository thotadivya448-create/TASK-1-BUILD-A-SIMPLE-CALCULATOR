# TASK-1-BUILD-A-SIMPLE-CALCULATOR
# DESCRIPTION
This project implements a simple command-line calculator using python.The calculator provides four fundamental arithmetic operations like addition,subtraction,multiplication,division the simple calculator is a command line application developed using python 3.x the calculator processes that user input,performs the selected calculation and displays the result it also includes error handling to manage invalid inputs and prevent division by zero errors,ensuring smooth execution of the program.

# OBJECTIVES
The objective of this project is to develop a simple command line calculator using python that performs basic arithmetic operation such asaddition,subtraction,multiplication, division the project helps users perform calculations quickly while demonstrating fundamental python programming concepts
